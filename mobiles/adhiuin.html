Adhi UIN Walisongo - Android App

Adhi UIN Walisongo is an Android Application developed for PT. Adhi Karya (Persero) Tbk - Gedung Department - UIN Walisongo Semarang construction project. This App was use to help construction process like Shop Drawing distributions, Material Stocks Management, etc.

Adhi UIN Walisongo is developed with Java (Android). Adhi UIN Walisongo App is no longer in use and has been pull out from Google Play Store because the Construction Project has Finished.
